## ✅ Inclusion Criteria

For a project to be listed in the main directory, it must:

- Be **publicly available** on GitHub with a working URL
- Have a **README** explaining installation and basic usage  
- Be **open source** with an identifiable license
- Be **meaningfully related** to scientific research or the OpenClaw ecosystem
- Be **actively maintained** or clearly labeled if stale
- Not be an **exact copy** of an already-listed project without clear differentiation

Projects that don't yet meet the bar may be added to the **Watching List** until they mature.

## ⏱️ Review Timeline

Submissions are reviewed weekly. You'll receive a comment on your issue when:
- ✅ The project is **approved and added** to the directory
- 🔄 **Changes are requested** (missing info, unclear description)
- ❌ The project is **declined** with a reason

The directory (`js/projects.js` in the [main repo](https://github.com/bajwa82mna/claw4science)) is updated after each review cycle.

## 🧬 About Claw4Science

Claw4Science tracks AI agent projects built on or inspired by the OpenClaw ecosystem, with a focus on scientific research applications — bioinformatics, biomedicine, drug discovery, and autonomous research systems.
